# Validator Transparency Dashboard (starter)

A clean, beginner-friendly Solana validator dashboard:
- Trust Card (commission, uptime snapshot, Jito on/off)
- Commission history sparkline
- UA/EN ready
- Mock data by default; flip to live data later

## Quick Start
1. Clone or download this repo.
2. Run with any static server or open `index.html` directly.
3. To use live data later, open `app.js` and set `USE_LIVE=true`, then add your keys in `.env.example` â†’ `.env`.
