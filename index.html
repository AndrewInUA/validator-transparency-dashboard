Source model. Live inputs come from Solana RPC + Stakewiz + Trillium + Jito proxy. Stability uses backend snapshot history plus current public inputs.
Methodology. Main stability ring uses all stored snapshots. Highlighted secondary score uses the recent snapshot window. Recent voting consistency is short-window RPC context.
History model. Opening a validator can register it for backend tracking. Snapshots are collected server-side on a schedule and shared for all viewers (not browser-local).
