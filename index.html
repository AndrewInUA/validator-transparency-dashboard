Methodology. “Epoch performance (proxy)” is a relative consistency score derived from epochCredits returned by getVoteAccounts. The dashboard takes the last six entries, looks at how voting credits changed from epoch to epoch, and compares each epoch with the best recent one. Those relative values are averaged into a 0–100% score over the last five epochs.
