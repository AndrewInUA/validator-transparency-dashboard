Source model. Public blocks use external data. Local tracking is a separate browser-only layer.

Methodology. Voting consistency is a relative score derived from recent epochCredits returned by getVoteAccounts. For each recent epoch, the dashboard looks at how many vote credits were added compared with the previous epoch, then compares that result with the strongest recent epoch. Those relative values are converted into a 0–100% score. Higher values mean more consistent recent voting performance.
